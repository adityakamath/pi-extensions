# Pi Extensions

This repository contains a collection of extensions ported for use with the [pi-coding-agent](https://github.com/badlogic/pi-mono/tree/main/packages/coding-agent), by [Badlogic (Mario Zechner)](https://github.com/badlogic).

## Extensions

### [GSD (Get Shit Done) Extension](./gsd)
- Port of the original [Get Shit Done framework](https://github.com/gsd-build/get-shit-done) for pi-coding-agent.
- Complete spec-driven development and productivity system for Pi.

### [Superpowers Extension](./superpowers)
- Port of the original [Superpowers framework](https://github.com/obra/superpowers) for pi-coding-agent.
- Adds advanced tools, workflows, and productivity features to Pi.

### [Remote Control Extension](./remote-control)
- Adapted from the original [control extension](https://github.com/mitsuhiko/agent-stuff/tree/main/pi-extensions/control.ts) by [Mitsuhiko (Armin Ronacher)](https://github.com/mitsuhiko)
- Provides remote session control capabilities for Pi.

---

Each extension folder contains its own README and documentation.
