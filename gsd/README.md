# GSD (Get Shit Done) Extension for pi-coding-agent

Port of the original [Get Shit Done framework](https://github.com/gsd-build/get-shit-done) for [pi-coding-agent](https://github.com/badlogic/pi-mono/tree/main/packages/coding-agent).

## Overview
This extension brings the complete, spec-driven GSD productivity system to pi-coding-agent, enabling project management, structured workflows, and turbocharged development in your terminal.

- Productive, guided coding using GSD methodologies
- Tons of commands and automation
- Designed to keep you shipping and focused

## Original Project
- [gsd-build/get-shit-done](https://github.com/gsd-build/get-shit-done)

## Usage
1. Copy or install this extension in your pi-coding-agent extensions folder or as a package:
   ```bash
   pi package add /path/to/gsd
   /reload
   ```
2. Type `/gsd:help` in pi for command list and help!
