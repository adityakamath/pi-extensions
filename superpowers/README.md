# Superpowers Extension for pi-coding-agent

Port of the original [Superpowers extension](https://github.com/obra/superpowers) for use with [pi-coding-agent](https://github.com/badlogic/pi-mono/tree/main/packages/coding-agent).

## Overview
The Superpowers extension brings advanced tools, workflows, and productivity enhancements into pi-coding-agent, inspired by the Superpowers project.

- Augments pi-coding-agent with unique features
- Productivity shortcuts, command extensions, and smart helpers

## Original Project
- [obra/superpowers](https://github.com/obra/superpowers)

## Usage
1. Install or copy the extension into your pi-coding-agent extensions or as a package:
   ```bash
   pi package add /path/to/superpowers
   /reload
   ```
2. Explore new capabilities and check `/help` for included commands!
