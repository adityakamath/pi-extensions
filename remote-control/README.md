# Remote Control Extension for pi-coding-agent

Adaptation of Mitsuhiko's [pi-control extension](https://github.com/mitsuhiko/agent-stuff/tree/main/pi-extensions/control.ts) for [pi-coding-agent](https://github.com/badlogic/pi-mono/tree/main/packages/coding-agent).

## Overview
Adds remote management and session control to pi-coding-agent:
- Connect/manage remote pi sessions
- Coordinate actions across machines
- Foundation for advanced automation and remote workflows

## Origin & Attribution
- Inspired by and adapted from: [mitsuhiko/agent-stuff - control.ts](https://github.com/mitsuhiko/agent-stuff/tree/main/pi-extensions/control.ts)

## Usage
1. Add/copy this extension into your pi-coding-agent or install as a package:
   ```bash
   pi package add /path/to/remote-control
   /reload
   ```
2. Use provided commands or see docs for remote session controls
